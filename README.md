## 阅读器测试

### 已编写的测试套件

- 信息源列表
  - 列表不为空
  - 列表下每个信息源有`URL`属性
  - 列表下每个信息源有`name`属性
- 菜单按钮
  - 菜单元素默认是隐藏的
  - 菜单图标被点击时会切换可见状态
- 信息内容
  - 信息被正确返回和显示
- 切换信息源
  - 能够切换信息源，且信息内容会发生改变

### 测试结果

所有测试均通过，程序运行正常。
